# 🌾 SafraVanGogh

Esse é o meu projeto para a primeira etapa da competição! É um sistema simples de foco (Timer Pomodoro) combinado com uma lista de metas, voltado para o tema de **Agro Forte e Futuro Sustentável**.

---

## 🎨 Por que Van Gogh?

Eu não queria entregar uma tela cinza ou um design genérico que todo mundo faz. Por isso, juntei o tema do Agro com a minha pintura favorita: **"O Semeador" de Vincent van Gogh**. 

Na tela dele, o trabalho no campo de trigo debaixo daquele sol gigante mostra que cultivar a terra exige paciência e cuidado diário. No meu projeto, o tempo funciona do mesmo jeito: cada minuto de foco é uma semente para o seu dia render mais.

**O que eu fiz no código para trazer a arte dele:**
* **Fundo Animado:** Criei um degradê no CSS que se mexe sozinho, misturando o verde da plantação com o dourado do trigo.
* **Efeito de Tinta a Óleo:** Usei um filtro SVG (`feTurbulence`) para distorcer o fundo. Isso faz a tela parecer uma pintura de verdade e não uma imagem colada da internet.
* **Sementes de Luz:** As estrelas piscando no fundo representam sementes flutuando no campo.

---

## 🚀 O que já está funcionando (MVP)

- **Ciclo do Semeador:** O cronômetro de 25 minutos funciona perfeitamente via JavaScript (inicia, limpa o tempo e avisa quando o ciclo acaba).
- **Metas de Cultivo:** Uma lista com exemplos reais de tarefas do agro, já estilizada com bordinhas douradas.
- **Acessibilidade Real:** Adicionei tags `aria-label` e `aria-hidden` para que pessoas cegas consigam usar o site com leitores de tela sem problemas.
- **Funciona no Celular:** Usei Media Queries no CSS para que o tamanho das letras se ajuste sozinho em qualquer tela.
